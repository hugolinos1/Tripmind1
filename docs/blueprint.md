# **App Name**: TripMind

## Core Features:

- User Authentication and Profile Management: Enable user registration, secure login, logout, and management of personal profile settings including name, email, preferred language, and currency, with data securely stored in SQLite using custom session management with HTTP-only cookies.
- Trip Planning & Dashboard: Allow users to create detailed trip plans through a multi-step wizard by specifying multiple destinations, date ranges, and travel preferences. Provide a central dashboard for an overview and easy access to all created and ongoing trips.
- AI Itinerary Generation Tool: Utilize an AI-powered tool to intelligently generate comprehensive, day-by-day itineraries based on user-defined trip details and preferences, including geocoding locations via Nominatim and saving event data to SQLite.
- Daily Itinerary Editor & Map View: Offer a dedicated interface for organizing and modifying daily events within a trip, allowing manual creation, editing, deletion, and reordering, complemented by an interactive map view showcasing event locations and routes using React-Leaflet and OpenStreetMap. Includes automatic location propagation between consecutive days.
- AI Event Details Enrichment Tool: Employ an AI tool to automatically enrich existing event details with practical information such as opening hours, price ranges, insider tips, and relevant images pulled from external sources like Unsplash or Lorem Picsum, saving enrichments to SQLite.
- Document and Attachment Management: Enable users to securely upload, attach, and manage essential travel documents (e.g., tickets, reservations) directly to specific events, supporting various file types with size validation and storing metadata in SQLite.
- AI Destination Insight Tool: Provide an AI-powered tool that compiles and presents extensive destination-specific practical information covering local vocabulary, gastronomy, customs, transportation options, weather insights, key attractions, and emergency details.

## Style Guidelines:

- The overall theme is dark. The primary color, used for main accents and interactive elements, is a warm amber (`#f59e0b`). The secondary accent color is a vibrant red (`#ef4444`). The main background is a very dark slate (`#0f172a`), providing depth and contrast. Event-specific colors like amber for 'visit', rose for 'meal', sky for 'transport', emerald for 'accommodation', and purple for 'activity' (`#f59e0b`, `#f43f5e`, `#0ea5e9`, `#10b981`, `#a855f7` respectively) are used for visual distinction.
- Headline font: 'Space Grotesk' (proportional sans-serif) for its modern, tech-savvy, and distinctive character. Body font: 'Inter' (grotesque-style sans-serif) for its high legibility, neutral appearance, and versatility in conveying detailed information.
- Utilize modern, clean outlined icons from 'Lucide React', consistent with the UI. Specific icons like 'MapPin', 'Utensils', 'Bus', 'Home', and 'Star' will be used for different event types, alongside relevant icons for interests and accessibility options.
- Implement a 'glassmorphism' aesthetic with translucent elements, using `rgba(0,0,0,0.4)` and `backdrop-filter: blur(20px)` for main glass panels, and `rgba(255,255,255,0.05)` for lighter elements, all set against a dark gradient background. The layout emphasizes clean grids for presenting information. Responsive design is paramount, following mobile-first patterns with breakpoints at `sm: 640px`, `md: 768px`, `lg: 1024px`, and `xl: 1280px`. This includes elements like sidebar collapsing to bottom navigation on mobile, full-screen modals, and touch-friendly controls. Event cards will support smooth drag-and-drop reordering, and day selectors will utilize horizontal scrolling with snap behavior for intuitive navigation.
- Incorporate subtle animations for UI feedback, such as transitions during AI itinerary generation or event enrichment (loading overlays, spinners). Implement smooth drag-and-drop animations for event cards to enhance user experience when reorganizing daily schedules.
